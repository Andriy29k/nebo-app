Підключення до API

У проді nginx на цій же адресі віддає фронт і проксує API на /health, /ready, /items — поле залиште порожнім (браузер ходить на той самий http://…, без прямого IP бекенду). Прямий бекенд лише для локальної розробки: http://127.0.0.1:8000 (не https://, якщо Flask без TLS).
